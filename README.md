# Cash Control

Proyecto de base de datos desarrollado con MongoDB.

Este sistema permite registrar información financiera de los usuarios.

Funciones principales:

- Registro de usuarios
- Registro de ingresos
- Registro de gastos
- Clasificación de gastos por categoría
- Registro de metas de ahorro
- Simulación de correos obtenidos por scraping

Base de datos utilizada:

cash_control

Colecciones:

usuarios
ingresos
gastos
categoria_de_gastos
metas_ahorro
correos_scraping

También se implementaron consultas de agregación para calcular el total de gastos de un usuario.
